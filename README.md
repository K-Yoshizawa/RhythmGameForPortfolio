# RhythmGame (ポートフォリオ用コードアーカイブ)

Unityで制作したリズムゲームのプロジェクトから、C#スクリプトのみを抽出したリポジトリです。
（アセット類の権利関係およびファイルサイズへの配慮のため、コードのみを公開しています）

## 🎮 ゲーム概要
**(ゲーム名未定)**
音楽に合わせてノーツを叩く、クラシックスタイルの6鍵リズムゲームです。
製作者が趣味の音楽ゲームの練習を家でしたいという理由で製作しました。


* **制作期間:** 約1ヶ月 (2026年1月14日時点で制作中)
* **エンジン:** Unity 2020.3.16f1
* **担当箇所:** 企画・実装すべて

## ⚠️ 注意事項
* 本リポジトリはポートフォリオ閲覧用にコードのみを抽出したものです。
* 画像、音声、外部プラグイン（TextMesh Pro, DOTweenなど）は除外されているため、CloneしてもUnity上では動作しません。
